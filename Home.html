Pixinsight Javascript GUI

Pixinsight is written using the Qt cross-platform development system, with a core programming language of C++.

Pixinsight though has a scripting environment built in. The language for this is the SpiderMonkey version of Javascript. This does have hooks into the underlying Pixinsight GUI environment.

This allows a Pixinsight user to develop there own scripting flows, and custom GUI.

This interface though lacks from some documentation.

This website is an attempt to document some of the GUI widgets that a Pixinsight user has access to, and how to write javascript code to make use of them.

There is an assumption that the reader has some coding experience, and in particular Javascript.

The Overview page in the menu, should give a quick overview on how to access the scripting environment, while the Available Widgets entry, should list available widgets with some usage models and example. Where possible, a link to the original Qt C++ Class definitions of the Javascript widgets version is made for clarity.

The rational for me looking at this was that although the BatchPreProcessing script is useful, I have started to use the SuperBias type files, which does not fit into this flow. I also use a standard naming convention, so can use regex to define which files to process at any stage til integration, so was looking to automate my flow.

Note: I have included a series of examples . They are located in a subdirectory Pixinsight files, in the directory structure on this website, and can be accessed from the relevant Widget page.
