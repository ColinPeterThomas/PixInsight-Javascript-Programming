RadioButton

The radioButton gives the user a widget (or multiple widgets) that can be boolean. It is similar to the CheckBox.
Normally there are a series of radioButtons in which exclusivity is maintained: only one selection at a time can be made.

i.e.


C++ documentation can be located at QtDocumentation and PCL Documentation which does give some hints to underlying functionality

To deploy, it is placed inside a Dialog widget.

this.exampleRadioButton = new RadioButton (this );


The (this) will pass the parent object (the dialog widget) to the newly created object.

To initialise , or set the value via code use the checked property:

this.exampleRadioButton.checked = true ;


When clicked, the event onClicked is emited. The user can define a function to be run, when this event is observed.

this.exampleRadioButton.onClick = function()

{

//..do some stuff....

}

To extract the current value of the widget  the checked value can be read, which should be boolean:


if (this.dialog.example.RadioButton.checked)

{

console.writeln("INFO: RadioButton activated..");

} else {

console.writeln("INFO: RadioButton deactivated..");

}


An example of this can be located in the file exampleRadioButton.js
