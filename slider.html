Slider

The Slider gives the user a widget that values can be changed using a slider, with predefined values.

i.e.


C++ documentation can be located at QtDocumentation and PCL Documentation which does give some hints to underlying functionality

To deploy, it is placed inside a Dialog widget.

this.newSlider = new Slider( this );


The (this) will pass the parent object (the dialog widget) to the newly created object.

When setting up a Slider, you need to be able to defined a range (min and max values) and an initial value: These are set below:

this.newSlider.minValue = 0;

this.newSlider.maxValue = 100;

this.newSlider.value = 50 ;

We then may want to extract the value of the slider, and be aware when the value has changed. The onValueUpdated event can track changes, while the .value parameter is the way to access the current value. The example below shows this:


this.newSlider.onValueUpdated = function()

{

console.writeln("INFO: Slider Changed ..");

console.writeln("Value:" + this.value);

}

Note in the function, the value of the Slider is acquired through the .value parameter. As this is the function from the Slider, inside the function it can be accessed through this.value. The path this.dialog.newSlider.value can also be used (and is the route outside of the function)


An example of this can be located in the file exampleSlider.js
